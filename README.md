## Instalation

## Usage

To add the dependency to your project, simply do:

```
yarn add @trustfractal/sdk
```

## Development

### Setup

First, clone & setup the repository:

```
git clone git@github.com:trustfractal/fractal-sdk.git
cd fractal-sdk
yarn install
```

### Testing

To run the tests you can simply do:

```
yarn test
```
